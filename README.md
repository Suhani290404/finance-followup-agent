# Finance Credit Follow-Up Email Agent

## Overview
Finance Credit Follow-Up Email Agent is a Streamlit-based web application that automates invoice tracking and payment reminder generation.

The system:
- Uploads invoice CSV files
- Calculates overdue payments
- Classifies escalation stages
- Generates automated follow-up emails

## Features
- CSV invoice upload
- Overdue day calculation
- Escalation classification
- Dashboard metrics
- AI-generated payment reminder emails
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- Pandas

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
